# LinkedIn Banner Images

## Files Available

1. **linkedin-banner.svg** - Gradient background version (teal gradient)
2. **linkedin-banner-simple.svg** - Solid background version (solid teal)

## LinkedIn Banner Specifications

- **Company Page Banner:** 1192 x 220 pixels
- **Personal Profile Banner:** 1128 x 191 pixels
- **Format:** PNG or JPG (SVG needs to be converted)

## How to Use

### Option 1: Convert SVG to PNG Online
1. Visit: https://cloudconvert.com/svg-to-png
2. Upload `linkedin-banner.svg` or `linkedin-banner-simple.svg`
3. Set width: 1192px, height: 220px
4. Download the PNG file
5. Upload to LinkedIn

### Option 2: Use Browser to Export
1. Open the SVG file in your browser
2. Right-click → "Save image as" (if available)
3. Or use browser developer tools to export as PNG

### Option 3: Use Image Editor
1. Open SVG in Adobe Illustrator, Inkscape, or Figma
2. Export as PNG at 1192x220 pixels
3. Upload to LinkedIn

## Upload to LinkedIn

1. Go to your LinkedIn Company Page
2. Click "Edit page" → "Edit cover image"
3. Upload the PNG file
4. Adjust positioning if needed
5. Save
